## Implementation Notes

### Slug Generation (pkg/utils/slug/slug.go)
```go
// go get github.com/gosimple/slug
func Generate(title string) string {
    return slug.Make(title) // "Rumah Mewah Jakarta" → "rumah-mewah-jakarta"
}
func GenerateUnique(title string, existsFn func(string) bool) string {
    base := Generate(title)
    candidate := base
    for i := 2; existsFn(candidate); i++ {
        candidate = fmt.Sprintf("%s-%d", base, i)
    }
    return candidate
}
```

### Slug — Where it's called
- In `listingService.Create()`: call `slug.GenerateUnique(req.Title, repo.ExistsBySlug)` → set `listing.Slug`
- In `listingService.Update()`: if `req.Title != nil` and title changed → regenerate slug

### ViewCount — Atomic SQL (MUST use this, NOT Save())
```go
// In repository:
func (r *listingRepository) IncrementViewCount(ctx context.Context, id uuid.UUID) error {
    return r.db.WithContext(ctx).
        Model(&entity.Listing{}).
        Where("id = ?", id).
        UpdateColumn("view_count", gorm.Expr("view_count + 1")).Error
}
```

### Owner Check (in service, NOT handler)
```go
func (s *listingService) checkOwnership(listing *entity.Listing, userID uuid.UUID, userRole string) error {
    if userRole == "admin" {
        return nil // admin bypass
    }
    if listing.UserID != userID {
        return domain.ErrForbidden
    }
    return nil
}
```

### Update — whitelist fields (MUST use Select, NOT Save)
```go
// In repository Update():
return r.db.WithContext(ctx).
    Model(listing).
    Select(fields). // explicitly whitelisted fields only
    Updates(listing).Error
```

### Route registration order (CRITICAL — slug before :id)
```go
// In router.go — slug route MUST be registered first
listingGroup.Get("/slug/:slug", listingHandler.GetBySlug) // ← first
listingGroup.Get("/:id", listingHandler.GetByID)          // ← second
```

### UUID parsing pattern (reuse in all handlers)
```go
id, err := uuid.Parse(c.Params("id"))
if err != nil {
    return fiber.NewError(fiber.StatusBadRequest, "invalid listing id")
}
```

### List query — NEVER use Joins() with has-many + Limit
```go
// CORRECT: use Preload for has-many associations
db.WithContext(ctx).
    Preload("Images").
    Preload("Category").
    Where(conditions).
    Limit(filter.Limit).Offset(offset).
    Find(&listings)
// Separate COUNT query for total:
db.Model(&entity.Listing{}).Where(conditions).Count(&total)
```

### Error handler — verify ErrForbidden → 403
Check the global error handler in `router.go` or `main.go`. It must include:
```go
case errors.Is(err, domain.ErrForbidden):
    return c.Status(403).JSON(...)
```
If missing, patch it.

## Migration 000003

### `000003_listing_crud_prep.up.sql`
```sql
-- 1. Fix price column: DECIMAL(18,2) → BIGINT (match int64 entity convention)
ALTER TABLE listings ALTER COLUMN price TYPE BIGINT USING price::BIGINT;

-- 2. Drop plain unique index on slug (blocks soft-delete reuse)
DROP INDEX IF EXISTS idx_listings_slug; -- check exact name from 000001
ALTER TABLE listings DROP CONSTRAINT IF EXISTS uni_listings_slug;

-- 3. Partial unique index: only enforce uniqueness for non-deleted rows
CREATE UNIQUE INDEX idx_listings_slug_active
    ON listings (slug) WHERE deleted_at IS NULL;

-- 4. Add CHECK constraint for status values
ALTER TABLE listings
    ADD CONSTRAINT chk_listings_status
    CHECK (status IN ('active', 'inactive', 'sold'));
```

### `000003_listing_crud_prep.down.sql`
```sql
ALTER TABLE listings DROP CONSTRAINT IF EXISTS chk_listings_status;
DROP INDEX IF EXISTS idx_listings_slug_active;
ALTER TABLE listings ADD CONSTRAINT uni_listings_slug UNIQUE (slug);
ALTER TABLE listings ALTER COLUMN price TYPE DECIMAL(18,2) USING price::DECIMAL(18,2);
```
